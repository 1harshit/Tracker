Problem

Unavailability of a low cost,secure,open source and easily configurable system for device tracking. Everyone likes to know that their product or any item that they are delivering has gone to the their destination place.

Solution

A low cost open source GPS based device tracking system which uses android devices to track and collect data.

Trackerstrap offers a accurate, reliable, and easy to use GPS tracking system.Internet based software allows the user to monitor and manage his device over the internet from anywhere via any device and present all these data in an easy to understand format.

This allows the user to track several type of resources at once.Rich user interface display a map with the current positions of the devices along with their paths for a pre-defined time.Trackerstrap provides the ability to filter data of different trackers and visualize them in a dashboard which is accesible by the manager.

User can easy attached a android device that runs Trackerstrap and track device via internet. Also user can monitor multiple devices at one time from this users time and money will save. With this Trackerstrap you ill feel safe. To enable this tracking system you will only need MAC Address of the device.

Once you add MAC Address of the device that you want to track you can see the tracking details on your Trackerstrap.

Features

  • Can add multiple devices.
  • User can add or remove device anytime.
  • Simple to use.
  • Rich UI design.
  • Easy deploy on service like EC2.

Developers

Tracker is to help developers to jump start developing any kind of mobility tracking applications.

This consists of several main modules

  • HTML5+AngularJS Front-end
  • NodeJS+MongoDB API for collecting and querying tracker data
  • Php+MySQL backend to manage the trackers & etc
  • Android app to push data to the API
  • Documentation
